
# Simple Learning Management System (LMS) - Prototype

This is a small static prototype of an LMS (front-end only) intended as a starter project.

**Contents**
- index.html — landing / courses list
- course.html — course detail template (static)
- admin.html — simple admin mockup
- assets/css/style.css — basic styling (responsive)
- assets/js/app.js — minimal client-side behavior (search, open course modal)
- data/courses.json — sample course data

**How to use**
- Unzip `lms-project.zip` and open `index.html` in your browser.
- No server required for the static prototype. For a dynamic backend, use any web server or integrate a simple API.
